# Screenshots Directory

This directory contains screenshots of the AI Prompt Firewall application.

## Screenshot Files

Place your screenshots here with the following naming convention:

- `demo-homepage.png` - Public demo UI homepage
- `demo-pii-detection.png` - PII detection example
- `demo-injection-block.png` - Injection blocking example
- `admin-logs.png` - Admin console logs tab
- `admin-policies.png` - Admin console policies tab
- `admin-rules.png` - Admin console regex rules tab
- `api-response.png` - API response example

## How to Add Screenshots

1. Take screenshots of your deployed application
2. Save them in this directory with the names above
3. Update `README.md` to include the screenshots
4. Commit and push to GitHub

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or 1280x720
- **Quality**: High quality, clear text
- **Content**: Show key features and functionality
